# 🧳 Brendan’s Developer Portfolio Vault

Welcome to my **Developer Portfolio Collection** — a one-stop repo where you can download tailored versions of my portfolio based on different roles:

- 🎨 Frontend Developer
- 🧠 Backend Developer
- 🧰 Full Stack Developer
- 📱 Mobile Developer

Each portfolio highlights relevant skills, tools, and projects specific to the role.

---

## 📂 What's Inside?
```
portfolio-vault/
├── frontend-portfolio/
│ └── frontend.pdf
│
├── backend-portfolio/
│ └── backend.pdf
│
├── fullstack-portfolio/
│ └── fullstack.pdf
│
├── mobile-portfolio/
│ └── mobile.pdf
│
└── README.md
```

Each folder contains:

- 🧑‍💻 A pdf resume
- 📱 A link to its portfolio website
- ✅ Projects relevant to that stack on the website
- 💡 A skills section (tech stack icons)
- 📬 Contact & Resume link
